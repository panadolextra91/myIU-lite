| Boundary | Description |
|----------|-------------|
| pushed code → CI runner | Untrusted/broken code reaches the gate; CI must catch failing tests/build before merge |

| job name → branch-protection required check | The `ci` job-name string is the contract plan 03 binds to; a rename silently breaks the gate |
