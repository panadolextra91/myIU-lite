## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → /admin/courses* API | Untrusted JSON; admin gate + server-side validation required |
| Scheduler → courses/audit tables | Automated mutation must attribute to SYSTEM and respect append-only audit |

## STRIDE Threat Register (ASVS L1, block on high)

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-14 | Tampering | resurrecting a soft-deleted course | mitigate | All reads filter deleted_at IS NULL; no un-delete endpoint this phase (D-29/D-40, Pitfall 5) |
| T-03-15 | Repudiation | sweep runs untraceable | mitigate | Non-empty sweeps audit-logged under SYSTEM actor (D-39); no-op days intentionally not logged, process visibility via app logs |
| T-03-16 | Spoofing | sweep actor spoofed/NULL | mitigate | systemID resolved via GetSystemUserID (D-38), passed explicitly to StartSweeper |
| T-03-17 | Denial of Service | sweep contention / runaway loop | accept | Single daily tick + startup run in single-instance binary; idempotent UPDATE; cancelable via ctx (D-37) |
| T-03-18 | Elevation of Privilege | non-admin course CRUD | mitigate | /admin/courses* behind AuthMiddleware + RequireRole(admin) |
| T-03-19 | Tampering | SQL injection via course filter/search params | mitigate | sqlc parameterized queries + sqlc.narg only |
| T-03-SC | Tampering | npm/shadcn install | mitigate | shadcn add copies source only; no new runtime deps |
</threat_model>
