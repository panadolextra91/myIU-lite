## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Browser → /admin/courses/:id/*/import API | Untrusted CSV + path course id; validate file, course-active, ID existence+role server-side |
| Uploaded ID CSV → server memory | Untrusted file content/size |

## STRIDE Threat Register (ASVS L1, block on high)

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-20 | Tampering | partial membership import on failure | mitigate | Two-phase validate-all-then-insert in one tx; all-or-nothing (D-27, Task 1) |
| T-03-21 | Tampering | enrolling into a soft-deleted course | mitigate | Target course validated active (deleted_at IS NULL) before any insert (Pitfall 5) |
| T-03-22 | Tampering | wrong-role / system / soft-deleted user enrolled | mitigate | GetUserIDsByRole validates role + is_system=FALSE + deleted_at IS NULL (Task 1) |
| T-03-23 | Denial of Service | oversized CSV upload | mitigate | http.MaxBytesReader + MaxMultipartMemory cap, 413 on oversize (Task 2) |
| T-03-24 | Tampering | SQL injection via CSV id / path params | mitigate | sqlc parameterized queries; ANY($1) binding; ids cast/validated as bigint |
| T-03-25 | Elevation of Privilege | non-admin import/removal | mitigate | All routes behind AuthMiddleware + RequireRole(admin) |
| T-03-26 | Spoofing | client-supplied actor on audit | mitigate | actor_id from c.Get("user_id") JWT claim (RESEARCH A4) |
| T-03-27 | Repudiation | silent CSV-driven removal hiding intent | mitigate | Removal is UI-only with per-action audit rows (D-43); no CSV-removal path exists |
| T-03-SC | Tampering | npm/shadcn install | mitigate | No new components vendored; no new runtime deps |
</threat_model>
